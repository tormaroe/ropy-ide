ROPY Documentation

Numerical operations

  • ADD (+)
  • SUBTRACT (-)
  • MULTIPLY (*)
  • DIVIDE (/)
  • MODULO (%)

ADD

Command symbol: +

Numerical operation; adds two numbers together and pushes the result on the stack.

Stack effect: ( n n -- n )

Example: 45+ ==> 9

SUBTRACT

Command symbol: -

Numerical operation; if the top element on the stack is a number A and the next element is a number B, pushes the result of subtracting B from A on the stack.

Stack effect: ( n n -- n )

Example: 26- ==> 4

MULTIPLY

Command symbol: *

Numerical operation; multiplies two numbers and pushes the result on the stack.

Stack effect: ( n n -- n )

Example: 45* ==> 20

DIVIDE

Command symbol: /

Numerical operation; if the top element on the stack is a number A and the next element is a number B, pushes the result of A divided by B on the stack.

Stack effect: ( n n -- n )

Example: 28/ ==> 4

MODULO

Command symbol: %

Numerical operation; if the top element on the stack is a number A (the dividend) and the next element is a number B (the devisor), pushes the remainder of the Euclidean division of A by N on the stack.

Stack effect: ( n n -- n )

Example: 48% ==> 0

Example 2: 38% ==> 2

Stack operations

  • SWAP (<)
  • DUPLICATE (>)
  • POP (?)

SWAP

Command symbol: <

Stack operation; Swaps the two topmost elements on the stack.

Stack effect: ( a b -- b a )

Example: 52<- ==> 3

DUPLICATE

Command symbol: >

Stack operation; Copies the topmost element and pushes it on the stack.

Stack effect: ( a -- a a )

Example: 2>+ ==> 4

POP

Command symbol: ?

Stack operation; Pops the topmost element off the stack.

Stack effect: ( a -- )

Example: 23? ==> 2
