Background

  • Entry into PLT Games December 2012: "Turing tarpit"

    "Beware of the Turing tar-pit in which everything is possible but nothing of interest is easy."
    -- Alan Perlis

  • Originally implemented in Ruby
  • Later in Common Lisp, Go, and JavaScript
  • Stack-based, esoteric, 2D programming language
  • Inspired by Befunge

Elements of a ROPY program

  • The Grid

  • The Stack

The Grid

  • 2D matrix of characters
  • Can be of any size
  • Any cell may contain a program instruction
